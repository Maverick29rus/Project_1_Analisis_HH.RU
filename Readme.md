# Project 1 — Анализ резюме из HeadHunter (Pandas)

![HeadHunter](https://raw.githubusercontent.com/AndreyRysistov/DatasetsForPandas/main/hh%20label.jpg)

## Описание задачи

Анализ и подготовка резюме, выгруженных с hh.ru, с целью исследования распределения зарплат и подготовки данных для дальнейшего моделирования. Основная цель — продемонстрировать практические навыки в извлечении данных, предобработке и исследовательском анализе данных с визуализацией результатов.

## Основные этапы

1. Скачивание и загрузка данных
2. Исследование структуры данных (размер, типы, пропуски)
3. Преобразование признаков (образование, пол/возраст, опыт, зарплата и др.)
4. Исследование зависимостей в данных (EDA, визуализации, зависимости зарплаты от должности/пола/образования)
5. Очистка и подготовка данных (обработка пропусков, выбросов, нормализация)

## Навыки и технологии

- Python, Jupyter Notebook
- Pandas, NumPy — обработка и преобразование данных
- Plotly — интерактивная визуализация
- Регулярные выражения и обработка текста
- Работа с CSV/архивами, загрузка данных из внешних источников
- EDA, предобработка данных и feature engineering
